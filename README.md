# Haier Evo Extended integration
[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/custom-components/hacs) [![stability-beta](https://img.shields.io/badge/stability-beta-33bbff.svg)](https://github.com/mkenney/software-guides/blob/master/STABILITY-BADGES.md#beta)

Форк оригинальной интеграции [and7ey/haier_evo](https://github.com/and7ey/haier_evo) с исправлением ряда ошибок надёжности.

Неофициальная интеграция для работы [Haier Evo](https://haieronline.ru/evo-iot/) (RU, KZ, BY).

Бета-версия: может работать некорректно.

## Отличия от оригинала

- **Исправлен баг** с ненадёжным включением кондиционера: команда могла теряться при обрыве WebSocket-соединения в момент отправки — теперь такая ситуация обнаруживается и команда отправляется повторно автоматически
- Команда `status=on` всегда включается при включении, независимо от кэшированного состояния
- Добавлен фолбэк для устройств, у которых API не возвращает стандартные маппинги статуса
- Улучшено логирование ошибок WebSocket и ответов устройства

Об ошибках и предложениях по оригинальной функциональности — в [issues оригинального репозитория](https://github.com/and7ey/haier_evo/issues).

## Поддерживаемые устройства
Ниже приведен список типов устройств, работа с которыми была **проверена пользователями**:

- Кондиционеры
- Холодильники

В действительности интеграция должна поддерживать больше устройств.

## Совместимость
Haier предлагает разные мобильные приложения для разных рынков и стран. Некоторые устройства совместимы с более чем одним приложением. Эта интеграция поддерживает только устройства, которыми можно управлять через приложение [Evo](https://haieronline.ru/evo-iot/). Загрузите приложение Evo и проверьте совместимость с вашим устройством.

## Установка

### 1. Установка HACS (если не установлен)
Для работы интеграции требуется наличие HACS.
Инструкция для установки https://www.hacs.xyz/docs/use/download/download/

### 2. Добавление репозитория Haier Evo Extended

Так как интеграция является кастомной, её необходимо добавить в HACS как пользовательский репозиторий:

1. Откройте **HACS** в боковой панели Home Assistant.
2. Нажмите на три точки (⋮) в верхнем правом углу и выберите **Custom repositories** (Пользовательские репозитории).
3. В поле **Repository** вставьте ссылку: `https://github.com/MaxSezin/haier_evo_extended`
4. В поле **Category** выберите **Integration** (Интеграция).
5. Нажмите **Add** (Добавить).
6. Теперь найдите **Haier Evo Extended** в поиске HACS и нажмите **Download** (Загрузить).

### 3. Настройка

1. Перезапустите Home Assistant.
2. Перейдите в **Настройки** > **Устройства и службы** > **Добавить интеграцию**.
3. Найдите в списке **Haier Evo** и введите данные вашего аккаунта от приложения Evo.
